# About ECHO

ECHO was created as part of a PhD dissertation at the University of North Dakota. The dissertation is titled "Echo: A Graphical User Interface Designed To Simulate, Study, And Mitigate Effects Of Interplanetary Communication Delays In Analog Missions" by Dario Gabriel Schor (August 2024).

You can find the dissertation here:

```
https://commons.und.edu/theses/6453/
```
